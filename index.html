HTML é a linguagem padrão usada para criar e estruturar páginas da web.
Ela consiste em uma série de elementos e tags que são usados para definir
estrutura e o conteúdo de uma página.

Tags

(Voltar)

As tags são o elemento fundamental do HTML. Elas são usadas para marcar diferentes partes de um documento e fornecer instruções sobre como essas partes devem ser exibidas. As tags geralmente têm uma tag de abertura < e uma tag de fechamento >, e o conteúdo é colocado entre elas.
Por exemplo, a tag <(P)> é usada para marcar parágrafos de texto, e o conteúdo do parágrafo é colocado entre as tags de abertura e fechamento.

Estrutura básica

(Voltar)

Um documento HTML começa com uma declaração chamada doctype, que informa ao navegador que tipo de documento HTML está sendo usado. Em seguida, vem a tag , que contém o cabeçalho <(head)> e o corpo <(body)> da página. O cabeçalho é usado para fornecer informações sobre a página, como o título e os estilos CSS, enquanto o corpo contém o conteúdo real da página.

Elementos de conteúdo

(Voltar)

HTML possui vários elementos para marcar diferentes tipos de conteúdo. Alguns exemplos incluem: h1 a h6 : tags de cabeçalho que representam diferentes níveis de título.

  • (p)>: tag de parágrafo para texto normal.
  • (a)>: tag de âncora usada para criar links.
  • (img)>: tag de imagem para exibir imagens.
  • (ul)> e (ol)>: tags para criar listas não ordenadas e ordenadas, respectivamente.
  • (table)>: tag para criar tabelas.

Atributos:

(Voltar)

As tags HTML também podem ter atributos que fornecem informações adicionais sobre o elemento. Por exemplo, a tag usa o atributo "href" para especificar o URL de destino do link. Os atributos são definidos dentro da tag de abertura e podem ter um valor atribuído a eles.

Comentários:

(Voltar)
Comentários em HTML são usados para adicionar notas ou descrições no código que não são exibidas no navegador. Eles são colocados entre os delimitadores <(!--e--)>.

Como fazer commits

(Voltar)

Depois de criar ou fazer alterações em um arquivo HTML, você pode fazer um commit usando um sistema de controle de versão, como o Git. Um commit é uma ação que registra as alterações feitas no arquivo em um repositório Git, permitindo que você mantenha um histórico de versões e reverta alterações, se necessário.

Para fazer um commit, você precisa inicializar um repositório Git no diretório do seu projeto usando o comando "git init". Em seguida, adicione o arquivo HTML ao rastreamento do Git usando o comando "git add nome-do-arquivo.html". Por fim, faça o commit das alterações usando o comando "git commit -m 'Mensagem do commit'".

Lembrando sempre de fornecer uma mensagem descritiva no comando de commit para que outros desenvolvedores (e eu mesmo) possam entender o propósito das alterações feitas no arquivo HTML.
